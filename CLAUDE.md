# Project

jpp - JSONPath processor

# Tech Stack

- Language: Rust (Edition 2024)

# Commands

- Build: `just build`
- Run: `just run`
- Test: `just test`
- Test with coverage: `just coverage`
- Lint: `just lint`
- Format: `just format`

# Branch Strategy

- Main: `main`
- Feature: `feature/<name>`
- Fix: `fix/<issue or description>`

# Commit Convention

Conventional Commits format:

- `feat:` New feature
- `fix:` Bug fix
- `docs:` Documentation
- `refactor:` Refactoring
- `test:` Tests
- `chore:` Other
